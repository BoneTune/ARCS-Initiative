# The ARCS Initiative

**ARCS (Agreement for Responsible Creative Stories)** is a set of guiding principles for Visual Novel development, with a focus on projects funded by the community. It is designed to foster trust, transparency, and accountability between creators and their patrons.

This repository contains the core documents of the initiative.

## Core Documents

*   **[ARCS Manifesto v1.0](./ARCS_Manifesto_v1.0.md)**
    *   The core list of 12 principles required for a project to be ARCS-compliant.

*   **[ARCS Commentary v1.0](./ARCS_Commentary_v1.0.md)**
    *   The official companion document that provides context, reasoning, and examples for each principle in the Manifesto.
